# Contributing

1. Create an issue before starting a large change.
2. Fork the repository and create a focused branch.
3. Keep Kodi-specific code behind the adapter layer where practical.
4. Add or update tests.
5. Run `python3 tools/verify.py` and `python3 -m pytest`.
6. Submit a pull request with a clear description and test notes.

All contributions are accepted under GNU GPL version 2.
